# Python SDK for Fibrous Finance Limit Orders

## Installation

**Create a Virtual Environment:**

```bash
python -m venv .venv
```

**Activate the Virtual Environment:**

- **Windows:**
```bash
.venv\Scripts\activate
```

- **Unix/MacOS:**
```bash
source .venv/bin/activate
```

**Install Required Packages:**
```bash
pip install aiohttp
pip install starknet_py
```
